# 📚 Unidade 1 - Introdução à Computação

## Objetivo

Nesta unidade foram apresentados os conceitos fundamentais da computação, sua evolução histórica e os principais componentes que formam um sistema computacional.

---

## Conteúdos

- História da Computação
- Gerações dos Computadores
- Sistemas Computacionais
- Hardware e Software
- Sistemas Binários

---

## Competências Desenvolvidas

- Compreender a evolução dos computadores.
- Identificar os principais componentes de um computador.
- Diferenciar hardware e software.
- Entender o funcionamento básico dos sistemas computacionais.

---

## Atividades

As atividades desta unidade foram desenvolvidas durante as aulas da disciplina e têm como objetivo reforçar os conceitos estudados.

---

## Conclusão

O estudo desta unidade fornece a base necessária para compreender os demais conteúdos relacionados à arquitetura de computadores e sistemas operacionais.
