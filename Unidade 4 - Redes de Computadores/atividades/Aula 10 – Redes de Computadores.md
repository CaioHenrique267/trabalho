# Startup Baseada em Sistemas de Informação

## SmartMarket

### Descrição da Startup

A SmartMarket é uma startup voltada para pequenos mercados e mercearias que enfrentam dificuldades para controlar estoques e prever a demanda de produtos.

A proposta da startup é utilizar dados gerados diariamente pelas vendas para transformar informações brutas em conhecimento estratégico. Com isso, os gestores conseguem tomar decisões mais inteligentes sobre compras, promoções e reposição de estoque.

O principal diferencial da SmartMarket é o uso de Inteligência Artificial para analisar padrões de consumo e prever tendências futuras, ajudando os estabelecimentos a reduzir desperdícios e aumentar seus lucros.

---

# 1. Problema e Dados (A Escolha do Caos)

## Problema

Muitos pequenos comerciantes realizam o controle de estoque manualmente ou com base apenas na experiência pessoal. Isso pode gerar:

- Falta de produtos importantes.
- Excesso de mercadorias com baixa procura.
- Desperdício de produtos perecíveis.
- Perda de vendas e lucro.

## Dados Coletados

A startup coleta os seguintes dados:

- Histórico de vendas.
- Quantidade de produtos vendidos.
- Horários de maior movimento.
- Datas comemorativas e feriados.
- Informações de estoque.
- Preferências dos clientes.

## Classificação dos Dados

### Entrada (Input)

- Registro de vendas.
- Cadastro de produtos.
- Dados dos clientes.
- Controle de estoque.

### Origem dos Dados

- Sistema de vendas (PDV).
- Aplicativo da startup.
- Banco de dados da empresa.

### Tipo dos Dados

#### Estruturados

- Nome do produto.
- Quantidade vendida.
- Data da venda.
- Preço.
- Código do produto.

#### Não Estruturados

- Avaliações dos clientes.
- Comentários.
- Sugestões recebidas pelo aplicativo.

---

# 2. Processamento (O Sistema)

A SmartMarket transforma dados em informações por meio de um Sistema de Informação composto pelas seguintes etapas:

## Coleta

Os dados são capturados automaticamente através do sistema de vendas e do aplicativo utilizado pelos clientes.

## Armazenamento

As informações são armazenadas em um banco de dados centralizado na nuvem.

## Processamento

Algoritmos organizam os dados, identificam padrões e calculam indicadores de desempenho.

## Análise

A plataforma gera dashboards, gráficos e relatórios que auxiliam os gestores na tomada de decisão.

## Tecnologias Utilizadas

- Banco de dados relacional.
- Computação em nuvem.
- Dashboards gerenciais.
- Inteligência Artificial.
- Algoritmos de previsão de demanda.

## Relação com os Elementos do Sistema

| Elemento | Função |
|-----------|---------|
| Sistema | Plataforma SmartMarket |
| Processos | Coleta, armazenamento e análise |
| Banco de Dados | Guarda todas as informações |
| Fluxo da Informação | Dados → Processamento → Relatórios |

---

# 3. Informação e Conhecimento (A Transformação)

Após o processamento dos dados, o sistema gera conhecimento por meio de:

## Padrões

- Produtos mais vendidos.
- Dias de maior movimento.
- Horários de pico.

## Tendências

- Crescimento ou queda nas vendas.
- Mudanças de comportamento dos clientes.

## Previsões

- Estimativa de vendas futuras.
- Necessidade de reposição de estoque.
- Produtos que terão maior demanda.

## Benefícios para o Usuário

Os gestores passam a ter informações precisas para planejar compras, promoções e investimentos, reduzindo riscos e aumentando a eficiência operacional.

---

# 4. Decisão e Valor (A Decisão)

## Decisões Permitidas

- Quando comprar novos produtos.
- Quais itens promover.
- Quanto manter em estoque.
- Quais produtos retirar do catálogo.

## Valor de Negócio

### Aumento da Eficiência

Automatiza análises que antes eram realizadas manualmente.

### Redução de Custos

Evita desperdícios e excesso de estoque.

### Personalização

Permite entender melhor as preferências dos clientes.

### Inovação

Utiliza Inteligência Artificial para gerar previsões e recomendações estratégicas.
